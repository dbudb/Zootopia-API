# Zootopia API

Generates an HTML website with animal data fetched from the API Ninjas Animals API.

## Setup

1. Clone the repo
2. `pip install -r requirements.txt`
3. Create a `.env` file with your API key:
   ```
   API_KEY=your_api_key_here
   ```

## Usage

```
python animals_web_generator.py
```

Enter an animal name and the program generates `animals.html` with the results.
